# Pomodomo 

![demo](./public/images/app-screenshot.png)

## Local development   

#### `Setup`
```shell
$ git clone https://github.com/khoaHyh/pomodomo.git

$ cd pomodomo

$ npm i
```

#### `Run React app dev server`
```shell
$ npm run dev
```

## Tech/Framework 

  * React (Next.js)
  * ChakraUI
  * Express
  * PostgreSQL
