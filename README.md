# Pomodomo frontend

![demo](./src/assets/app-screenshot.png)

## Local development   

#### `Setup`
```shell
$ git clone https://github.com/khoaHyh/pomodomo-frontend.git

$ cd pomodomo-frontend

$ npm i
```

#### `Run React app dev server`
```shell
$ npm start
```

## Tech/Framework used on frontend

  * React
  * ChakraUI
